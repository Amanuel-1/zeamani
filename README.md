
This is my personal website built with modern web technologies, showcasing my projects, blog posts, and ideas. Built with a focus on speed, scalability, and dynamic content.

## Tech Stack

- **Next.js**: Framework for building fast, server-rendered React applications.
- **Supabase**: Provides backend services like database, authentication, and real-time APIs.
- **Prisma**: ORM for working with databases in a type-safe way.
- **Sanity**: Headless CMS for managing content dynamically.
- **NextAuth**: Authentication for secure login and user sessions.

## Features

- **Dynamic Content Management**: Powered by Sanity, easily manage and update content.
- **Authentication**: Secured with NextAuth, enabling custom user sessions.
- **Fast and Scalable**: Built with Next.js, optimized for performance and SEO.
- **Responsive Design**: Mobile-friendly and visually appealing across devices.
